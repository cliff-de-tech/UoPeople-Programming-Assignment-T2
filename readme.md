# UoPeople Programming Assignment - Term 2

A collection of Python programming assignments for UoPeople's Programming Course, Term 2.

## Overview

This repository contains solutions for programming assignments focusing on:
- **Recursion**: Understanding recursive functions with countdown and countup examples
- **Error Handling**: Demonstrating proper exception handling in Python

## Repository Structure

```
.
├── readme.md
└── Unit 1/
    ├── README.md                      # Detailed documentation for Unit 1
    ├── countdown_countup.py           # Recursive countdown/countup functions
    ├── division_error_handling.py     # Interactive error handling demo
    ├── test_error_with_handling.py    # Error handling demonstration
    └── test_error_without_handling.py # Unhandled error demonstration
```

## Getting Started

### Prerequisites
- Python 3.x

### Running the Programs

Navigate to the `Unit 1` directory and run any of the Python files:

```bash
cd "Unit 1"

# Run the recursive countdown/countup program
python countdown_countup.py

# Run the interactive division error handling demo
python division_error_handling.py

# Run error handling demonstrations
python test_error_with_handling.py
python test_error_without_handling.py
```

## Units

### Unit 1: Conditionals and Recursion with Error Handling

This unit covers:
1. **Recursive Functions**: Implementation of `countdown` and `countup` functions that demonstrate recursion with positive and negative numbers
2. **Error Handling**: Multiple approaches to handling division by zero errors, including try-except blocks and preventive checking

For detailed documentation, see [Unit 1/README.md](./Unit%201/README.md).
